## Descripción

United Solar Family es realizada con ReactJs, Webpack y Materialize, con el patrón de diseño "glassmorphism" y con un Ingenio Innato.

## Instalación

```bash
$ npm install
```

## Correr la aplicación

```bash
# development
$ npm run start

# production mode
$ npm run start:prod
```
